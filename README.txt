# Plataforma de Gestión y Monitoreo Ambiental OpenAQ

## Descripción del Proyecto
Este proyecto es una interfaz web de maquetado para un Dashboard de Monitoreo Ambiental. Diseñado según los principios de Interacción Hombre-Máquina (HCI) y accesibilidad visual.

> **Nota:** Siguiendo las observaciones del proyecto, esta versión está construida **exclusivamente con HTML5 y CSS3**, dejando el espacio reservado para la posterior integración de mapas e interacciones mediante APIs.

## Estructura de Archivos
- `index.html`: Estructura semántica del Dashboard (Header, Menú lateral, Tarjetas de estado, Espacio para el Mapa, Tabla de sensores y Panel de detalles).
- `styles.css`: Hojas de estilo responsables de la maquetación en Grid y Flexbox, paleta de colores y simulación de gráficos mediante reglas CSS pura.
- `README.md`: Documentación del proyecto.

## Características de Diseño
- **Diseño Modular:** Organizado mediante CSS Grid en tres secciones principales (Sidebar, Dashboard central y Panel de detalles).
- **Interfaz Intuitiva:** Uso de jerarquía visual clara y componentes reconocibles (cards, datos tabulares y barras de progreso/gráficos).
- **Cumplimiento de Restricciones:** Cero dependencias de JavaScript o frameworks externos.
